# Security Policy

## 安全掃描

所有技能在發布前經過四維度審計：

| 維度 | 檢查內容 |
|------|---------|
| D1 設計質量 | Frontmatter、文件組織、Token 效率 |
| D2 內容質量 | 觸發條件、執行步驟、錯誤處理 |
| D3 安全性 | SQL 注入、密碼存儲、權限控制、危險模式 |
| D4 實用性 | 輸出格式、使用場景、依賴說明 |

## 已知風險

- **可執行腳本**：部分技能包含 .py/.ts/.js 腳本，安裝前建議檢查
- **第三方依賴**：@automatelab 系列包含完整 MCP server 代碼
- **無代碼簽名**：zip 文件未簽名，建議比對 SHA256

## 安裝建議

1. 只安裝你需要的技能
2. 安裝前檢查 SKILL.md 和腳本內容
3. 使用 `scripts/` 目錄時注意權限
4. 定期更新到最新版本

## 報告安全問題

如果發現安全問題，請在 GitHub Issues 中報告。
